clj-foundation.conversions

A ‘convert’ multimethod that can convert between arbitrary types. Default implementations are supplied for Clojure’s built-in types.

convert

multimethod

Convert src-instance to dest-class if possible. Returns patterns/NO-RESULT-ERROR on failure. For examplle:

  • (convert Boolean/TYPE “true”)
  • (convert Map vararg-parameter-kvs)
